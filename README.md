Lua-iOS-Tutorial
================

在iOS中使用Lua的一个教学Demo

###Usage

使用命令
  
	git clone https://github.com/ziggear/Lua-iOS-Tutorial.git 
 
复制库到本地，然后通过check相应版本的tag，来获得对应的代码Demo  
例如：

	git checkout t1
